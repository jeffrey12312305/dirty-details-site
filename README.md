# Dirty Details LLC Website

React + Vite starter site for a mobile detailing business.

## Run locally

npm install
npm run dev

## Build

npm run build

## Edit business info

Open src/data.js and update:

- Business name
- Phone
- Email
- City
- Social links
- Services
- Packages
- Testimonials
- Blog posts

## Deploy to Vercel

Import the GitHub repo into Vercel.

Framework preset: Vite
Build command: npm run build
Output directory: dist
